kg_microbe.download.download(yaml_file: str, output_dir: str, ignore_cache: bool = False) → None¶

Downloads data files from list of URLs (default: download.yaml) into data directory (default: data/).

Parameters

  • yaml_file – A string pointing to the yaml file utilized to facilitate the downloading of data.

  • output_dir – A string pointing to the location to download data to.

  • ignore_cache – Ignore cache and download files even if they exist [false]

Returns

None.

kg_microbe.transform module¶

kg_microbe.transform.transform(input_dir: str, output_dir: str, sources: List[str] = None) → None¶

Call scripts in kg_microbe/transform/[source name]/ to transform each source into a graph format that KGX can ingest directly, in either TSV or JSON format: https://github.com/NCATS-Tangerine/kgx/blob/master/data-preparation.md

Parameters

  • input_dir – A string pointing to the directory to import data from.

  • output_dir – A string pointing to the directory to output data to.

  • sources – A list of sources to transform.

Returns

None.
